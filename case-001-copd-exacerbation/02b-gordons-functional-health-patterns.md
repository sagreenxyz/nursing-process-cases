# Data Analysis Using Gordon's Functional Health Patterns

## Overview

**Gordon's Functional Health Patterns** is a holistic nursing assessment framework developed by Marjory Gordon. It organizes patient data into 11 functional patterns that represent the whole person - not just body systems.

This framework is particularly useful because:
- ‚úì It's **person-centered**, not disease-centered
- ‚úì It captures **psychosocial** aspects often missed in body systems reviews
- ‚úì It aligns with **NANDA-I nursing diagnoses** (many diagnoses map directly to patterns)
- ‚úì It helps identify **strengths** as well as problems
- ‚úì It considers the patient's **lifestyle, culture, and values**

## When to Use Each Framework

### Use Body Systems Clustering When:
- Acute medical crisis requiring rapid physiological assessment
- Medical-surgical focus
- Communicating with physicians (they think in systems)
- Prioritizing physiological problems quickly

### Use Gordon's Functional Health Patterns When:
- Comprehensive nursing assessment
- Chronic disease management
- Discharge planning (identifies barriers)
- Holistic care planning
- Understanding the person's life context
- Identifying psychosocial needs

### Best Practice:
**Use BOTH.** Start with systems in acute crisis, then expand to patterns for comprehensive assessment.
